Cuaderno Personal

Semana 1

¿Qué temas aprendí?

Fundamentos de las tecnologías web

En la primera semana del curso, aprendí una serie de conceptos introductorios esenciales. Comencé comprendiendo la importancia del protocolo TCP/IP en la comunicación en línea y cómo facilita la transferencia de datos entre dispositivos en una red. Además, aprendí sobre el papel fundamental del protocolo HTTP en la World Wide Web, permitiendo la transferencia de información y recursos entre servidores y clientes.

Dentro del desarrollo de aplicaciones web, se discutieron los diferentes roles en el proceso. Se destacaron dos roles principales: el desarrollador Frontend y el desarrollador Backend. Para el lado del cliente o Frontend, se trató de las tecnologías clave, como HTML, JavaScript y CSS, y cómo se utilizan para crear la interfaz de usuario de una aplicación web. También se exploraron herramientas importantes, como Node.js y TypeScript, que mejoran la eficiencia en el desarrollo web. Además, se mencionaron ejemplos de frameworks JavaScript, como Angular, que simplifican la creación de aplicaciones web interactivas.

En el lado del servidor o Backend, se introdujeron varios lenguajes de programación utilizados, como PHP, Java, Python y Ruby, cada uno con sus propias ventajas y casos de uso. También se abordaron diferentes gestores de bases de datos, como SQL Server, Oracle, SQLite, MySQL y MariaDB, junto con consideraciones al elegir uno para una aplicación web. Por último, se destacó la importancia del diseño de la interfaz de usuario (UI) y la experiencia del usuario (UX), y se presentaron herramientas como Figma y Adobe XD para crear prototipos y diseños de aplicaciones web.

Reflexión

ha sido un sólido punto de partida. He adquirido conocimientos fundamentales sobre protocolos web, roles en desarrollo de aplicaciones y tecnologías esenciales

Semana 2

Diseño Creativo

¿Qué temas aprendí?

Lenguaje Marcado de Hipertexto HTML y Lenguaje Marcado Extensible XML, uso css

Durante esta semana, aprendí temas fundamentales relacionados con HTML y un poco de CSS. Comencé entendiendo la "Open Web Platform", una colección de tecnologías abiertas y estándares como HTML5, CSS, SVG y MathML. Esta plataforma promueve principios como la independencia de las empresas y la transparencia, lo que la convierte en la base sólida del desarrollo web moderno. Además, adquirí un entendimiento profundo de HTML, un lenguaje de marcado que define la estructura y semántica del contenido web, desde sus etiquetas fundamentales hasta su historia como una herramienta de hipertexto.

En cuanto a procedimientos, profundicé en la construcción de páginas web mediante HTML y CSS. Aprendí cómo funcionan estos lenguajes y cómo utilizarlos para estructurar y estilizar el contenido web. Exploré las etiquetas principales en HTML y las unidades de longitud utilizadas en el diseño web. Además, me adentré en el mundo de los selectores y las técnicas de maquetación en CSS, incluyendo conceptos avanzados como Flexbox y Grid, así como técnicas de posicionamiento y diseño fluido. También abordé prácticas esenciales de accesibilidad y me familiaricé con las normativas estándares, como las pautas WCAG 2.1, para garantizar una experiencia web inclusiva.

Obtuve un conocimiento del Document Object Model (DOM) y cómo se representa como un árbol de elementos en JavaScript. Esto me permitió comprender cómo interactuar dinámicamente con el contenido y los estilos de una página web. También exploré el ciclo de vida de una página web, desde su carga inicial hasta su renderización y visualización en el navegador. Desarrollé habilidades sólidas para crear páginas web efectivas, responsivas y accesibles, lo que me preparó para abordar proyectos web de manera exitosa en el futuro.

Reflexión

Descubrir la importancia de estándares abiertos y la accesibilidad en la creación de sitios web me ha hecho apreciar la responsabilidad que conlleva el desarrollo web. Adquirir conocimientos prácticos en HTML, CSS y la manipulación dinámica de elementos en la web me ha dado la confianza para enfrentar desafíos.

Semana 3

Diseño Creativo

¿Qué temas aprendí?

Front-End con Librería CSS Bootstrap y Tailwind

Durante esta fase del curso, aprendí la creación de páginas web Front-End utilizando dos destacadas librerías de CSS: Bootstrap y Tailwind. En primer lugar, exploré Bootstrap, una librería ampliamente utilizada que simplifica la creación de interfaces de usuario atractivas y responsivas. A través de este estudio, comprendí cómo aprovechar las herramientas proporcionadas por Bootstrap para diseñar diseños de página efectivos y componentes reutilizables. Además, aprendi del uso dee Tailwind, otra librería de CSS que se caracteriza por su enfoque modular y personalizable en el desarrollo de interfaces de usuario.

En términos de procedimientos, aprendí cómo crear y personalizar diseños de página web utilizando ambas librerías. Esto incluyó el manejo de conceptos como contenedores, columnas, puntos de quiebre y la disposición de elementos en la página. También profundicé en la tipografía, abordando aspectos como las fuentes, líneas y efectos de texto, y exploré cómo aplicar estilos a fondos, bordes y tablas.

Además, me sumergí en el uso de Flexbox, una técnica de diseño que permite una alineación y distribución flexible de elementos en la página. Esto incluyó conceptos como la dirección de flexión, el ajuste de elementos y el crecimiento de elementos en relación con el espacio disponible. Por último, pero no menos importante, aprendí a diseñar la interfaz de usuario de aplicaciones web, abordando elementos comunes como encabezados, visualización de datos, listas, formularios, navegación y superposiciones. Esto me proporcionó las habilidades necesarias para crear experiencias de usuario efectivas y atractivas en aplicaciones web utilizando Bootstrap y Tailwind, aprovechando al máximo sus capacidades para diseñar interfaces de usuario responsivas y personalizables. Como parte práctica se desarrolló diseños responsivos usando Flexbox, grid y float

Reflexión

He aprendido a personalizar diseños, tipografía y efectos de estilo, además de usar Flexbox para una distribución flexible de elementos. También he mejorado mis habilidades en el diseño de interfaces de usuario efectivas.

Semana 4

Diseño Creativo

¿Qué temas aprendí?

Manejar el DOM con Javascript y Canvas

Comencé viendo los motores JavaScript, las variables y los operadores, lo que me permitió entender cómo se ejecuta y procesa el código JavaScript en los navegadores. Además, profundicé en los tipos de datos, como los métodos y booleanos, y aprendí a utilizar estructuras de control y bucles para gestionar el flujo de ejecución en mis programas.En el ámbito de las funciones, adquirí un entendimiento de su sintaxis y cómo invocarlas. Exploré varios tipos de funciones, desde las anónimas hasta las funciones flecha.

Otra área importante de aprendizaje fue el manejo del Document Object Model (DOM) con JavaScript y Canvas. Esta habilidad me permitió interactuar con la estructura y el contenido de una página web de manera dinámica, lo que es esencial para la creación de aplicaciones web interactivas y dinámicas. También se destacó la importancia de JavaScript como un lenguaje de scripting dinámico en el desarrollo web moderno. Además, se presentó Node.js como una plataforma que amplía las capacidades de JavaScript al permitir su ejecución en el lado del servidor, lo que abre nuevas posibilidades en el desarrollo web.

Durante la práctica, apliqué estos conocimientos creando elementos visuales interactivos en Canvas, como un reloj analógico y un gráfico de pastel. Estos ejercicios prácticos fortalecieron mi comprensión de JavaScript y su aplicación en la creación de experiencias interactivas en el navegador.

Reflexión

Este aprendizaje en JavaScript ha sido revelador. Desde los fundamentos hasta la manipulación del DOM y la introducción a Node.js, he descubierto un amplio campo de posibilidades en el desarrollo web

Semana 5

Diseño Creativo

¿Qué temas apredió?

Desarrollo de aplicación FrontEnd con Framework JS

Aprendi la historia de React, lo que me proporcionó un contexto valioso sobre su evolución y su relevancia en la industria actual. Además, me sumergí en un conjunto de herramientas esenciales para trabajar con React, como Babel.js, ESLint y Beauty Tools/CSS-Beautifier.php. Estas herramientas facilitan el desarrollo y el mantenimiento del código, asegurando una práctica óptima en la creación de aplicaciones React.

Para la gestión de recursos y estilos en aplicaciones React, adquirí conocimientos sobre FontAwesome.com y Favicon.com, que son recursos valiosos para el manejo de iconos y favicons en sitios web. Además, exploré Webpack.js, una herramienta de construcción (bundler) que simplifica la gestión de dependencias y mejora el rendimiento de las aplicaciones React.

En el ámbito de los estilos, profundicé en el uso de SCSS y SAS, que son preprocesadores que permiten una gestión más eficiente y modular de los estilos en aplicaciones web. También me familiaricé con herramientas de instalación y configuración esenciales como Vite.js, Remixram, Next.js y Gatsby.js, que son ampliamente utilizadas en el desarrollo de aplicaciones React.

Reflexión

Esta inmersión en React ha sido un viaje educativo y emocionante que me ha equipado con las habilidades y conocimientos necesarios para desarrollar aplicaciones web más sofisticadas y eficientes.

Uso de Hooks

Aprendí a utilizar diversos hooks en React, como useState para gestionar el estado, useEffect para operaciones secundarias, useContext para compartir datos, useRef para interactuar con el DOM, useReducer para manejar estados complejos, useCallback y useMemo para optimizar funciones y cálculos. También exploré la creación de hooks personalizados, mejorando la eficiencia y modularidad de mis componentes en React. Este conocimiento ha mejorado mi habilidad para escribir código más limpio y mantenible en proyectos React.

Reflexión

Explorar los hooks en React ha optimizado mi gestión del estado y lógica de componentes. Desde useState hasta useReducer, cada hook ofrece soluciones poderosas. Crear hooks personalizados amplió mi capacidad para construir componentes eficientes en solo tres líneas.
